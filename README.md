# Group7
Software Assignment 1

This project focuses on preparing positive and negative test cases for two important modules of the Flipkart web application: Order History and Profile Management.
Testing these modules is essential to ensure that the system functions correctly under both valid and invalid user conditions. The test cases evaluate how the system behaves when users perform common actions such as viewing previous orders, updating personal information, or attempting invalid operations.
The objective of this project is to apply software testing techniques to identify potential issues and verify that the application meets functional requirements.
